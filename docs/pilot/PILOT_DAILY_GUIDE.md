# Pilot Daily Guide

## Daily flow

1. Unlock iMOS.
2. Review daily briefing.
3. Confirm current priority.
4. Review commitments and decisions.
5. Review Rosie recommendations.
6. Accept, reject, or correct Rosie as needed.
7. Complete evening reflection.
8. Open Pilot dashboard.
9. Save daily pilot check in.

## Daily check in focus

1. Briefing usefulness.
2. Recommendation usefulness.
3. Cognitive effort.
4. Missing context.
5. Incorrect assumptions.
6. Trust concerns.
7. Decision clarity support.
8. Mission advancement support.

## Notes

1. Keep comments optional to reduce effort.
2. Keep check in concise.
3. Use concern review when warnings appear.
4. Confirm backup readiness when prompted.
