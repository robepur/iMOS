# Pilot Completion Review

## Evidence based verdict options

1. Continue to production preparation.
2. Extend pilot.
3. Pause and review.
4. Rosie refinement required.
5. Security review required.
6. Recovery review required.

## Review inputs

1. Active day count.
2. Remaining day count.
3. Workflow completion trend.
4. Briefing usefulness trend.
5. Recommendation usefulness trend.
6. Cognitive effort trend.
7. Correction count.
8. Missing context reports.
9. Trust concern reports.
10. Unresolved concern count.
11. Backup readiness.

## Required operator decision

Operator is always the final decision maker. Build 026 provides evidence, not autonomous pilot conclusions.

## Exit checks

1. Confirm unresolved concerns.
2. Confirm backup readiness.
3. Export local report if required.
4. Record final operator decision.
