# Build 026 Pilot Activation

## Activation requirements

1. Onboarding completed.
2. Recovery backup confirmed.
3. Vault unlocked.
4. Local only operation available.
5. Synchronization remains optional and disabled by default.
6. Operator acknowledges pilot purpose.
7. Operator acknowledges private local measurement.
8. Operator selects duration (default 30, minimum 14).
9. Operator explicitly starts pilot.

## Activation path

1. Open More menu.
2. Select Pilot.
3. Create pilot session.
4. Confirm acknowledgements.
5. Start pilot.

## Pilot controls

1. Pause pilot.
2. Resume pilot.
3. Extend pilot.
4. End early.
5. Complete pilot.
6. Export report.
7. Delete pilot measurements.

## Guardrails

1. No operator score.
2. No operator grade.
3. No operator ranking.
4. No automatic Rosie authority expansion.
5. No pilot data transmission.
