# iMOS

**Individual Mission Operating System**

A private first, standalone personal operating environment built for one operator.

## Mission

iMOS helps the operator stay ahead of reality through trusted observation, disciplined reasoning, purposeful execution, and a lasting partnership with Rosie.

## Current build

Build 001 establishes the first daily operating loop:

1. Arrival
2. Rosie briefing
3. Focus session
4. Reflection

## Product boundaries

• Personal use only
• No ARGUS connection
• No multi tenant architecture
• No external integrations in Build 001
• No personal data committed to source control

## Local development

```bash
npm install
npm run dev
```

## Security

This repository must not contain secrets, credentials, personal records, calendar data, email content, or private operator memory.
